Summary

This project demonstrates a utilization of Python script to visualize the weather of 500+ cities across the world of varying distance from the equator. In addition to Python script, this project required using the Matplotlibrary, CityPy and the OpenWeatherAPI.

After randomly assigning Lats/Longs, the OpenWeatherMap API was used to take weather data for 500+ cities. The data from theses cities provided the basis for the Weather Data Project.

The dataframe was saved to a csv and the data was susbsequently analyzed even further by the equator (seperating latitude by the Northern and Southern Himsphere). To showcase the relationship between latitude and temperature/humidity/cloudiness/wind speed I utilize MatplotLib scatter plots to view:

  • Temperature (F) vs. Latitude
  • Humidity (%) vs. Latitude
  • Cloudiness (%) vs. Latitude
  • Wind Speed (mph) vs. Latitude
